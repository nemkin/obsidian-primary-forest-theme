# Primary Forest for Obsidian

A forest green color variant of the beautiful [Primary for Obsidian](https://github.com/primary-theme/obsidian) by [Cecilia May](https://github.com/ceciliamay).

Replaces the original warm amber/sandy palette with deep forest greens, evoking the feel of a pine forest or jungle canopy, while preserving the full depth and structure of the original theme.

If you enjoy Primary, please support Cecilia May's work at the [original repository](https://github.com/primary-theme/obsidian).
